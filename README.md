# 🚀 Asteroid Data Analysis & Prediction

A Streamlit web application for analyzing asteroid data and predicting:

- 🌍 Asteroid Diameter
- ☄️ Potentially Hazardous Asteroid (PHA)

The application uses trained Machine Learning models hosted on Hugging Face for efficient deployment.
